A-Frame Geo Projection Component Example Gallery

aframe-geo-projection-component is a component used for creating VR maps in A-Frame. It uses D3.js's d3-geo library to turn geojson or topojson into entities in VR.

  • Lines

    A map of the world projected onto a plane using a stereographic projection.

  • Shapes

    A map of the world projected onto a plane using a stereographic projection where the land is rendered as flat shapes.

  • Extruded

    A map of the world projected onto a plane using a stereographic projection where the land is rendered as extruded 3D shapes.

  • TopoJSON

    A map of the USA using TopoJSON (pre-projected using geoAlbersUSA) where the states are rendered as lines.

  • Selection

    An example of how to use the geo-selection component to highlight the state that the person is looking at.

  • U.S. Population by State in 2017

    A visualization of the estimated U.S. Population by State as of June 1, 2017.

    Data: Table 1. Annual Estimates of the Resident Population for the United States, Regions, States, and Puerto Rico: April 1, 2010 to July 1, 2017 (NST-EST2017-01)
    Source: U.S. Census Bureau, Population Division
    Release Date: December 2017
